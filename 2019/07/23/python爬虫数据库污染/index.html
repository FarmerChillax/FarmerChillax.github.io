工作原理是不断的模拟用户注册从而污染数据库。

前言

前置条件

  • python3
  • 需要用到python的模块(库)
    • requests
  • chrome浏览器

实现思路

  1. 利用浏览器的开发者工具查看后端接口
  2. 分析后端接口
  3. 利用random库随机生成字符做用户名和密码
  4. 利用爬虫进行数据提交
